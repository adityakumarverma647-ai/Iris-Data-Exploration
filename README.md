# Iris Data Exploration Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Iris Dataset using Python. The goal is to understand the dataset's structure, identify patterns, analyze feature relationships, and visualize data using various statistical and graphical techniques.

## Dataset Information

The Iris dataset contains 150 samples of iris flowers from three species:

* Setosa
* Versicolor
* Virginica

Features included:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Project Workflow

1. Import required libraries
2. Load the Iris dataset
3. Explore dataset structure
4. Check for missing values
5. Generate statistical summaries
6. Create visualizations
7. Analyze correlations and patterns
8. Draw conclusions

## Visualizations

* Dataset Preview
* Missing Value Analysis
* Histogram
* Boxplot
* Correlation Heatmap
* Pairplot
* Species Distribution Chart

## Key Findings

* No missing values were found.
* Petal Length and Petal Width show strong positive correlation.
* Setosa species is clearly distinguishable from the other species.
* The dataset is balanced with 50 samples per species.
